## 1. 문제 정의

### 1.1 배경

한글 타자 연습은 PC 사용의 기초 스킬이지만, 기존 웹 기반 타자 연습 사이트는 단순 반복 입력 방식으로 지루함을 유발한다. 특히 초보자나 어린 학습자는 흥미를 잃고 중도 포기하는 경우가 많다.

### 1.2 해결하려는 핵심 문제

- 타자 연습이 단조로워 지속 학습이 어렵다
- 실력 향상 피드백이 즉각적이지 않아 동기 저하
- 자신의 타이핑 속도/정확도를 체계적으로 파악하기 어렵다

## 2. 요구사항

### 2.1 기능 요구사항 (Functional Requirements)

#### FR-01: 게임 화면 (핵심)
- 게임 화면 상단에서 한글 낱말이 무작위로 생성되어 아래로 낙하
- 낱말이 화면 바닥에 닿기 전에 키보드로 정확히 입력하면 제거
- 현재 입력 중인 단어와 매칭되는 낙하 단어를 하이라이트 표시
- 입력창은 하단 고정, 한글 조합 입력 정상 처리

#### FR-02: 레벨 스테이지 시스템
- 레벨 1~10 존재 (총 10 스테이지)
- 각 레벨 클리어 조건: 목표 점수 달성 (레벨 × 100점)
- 레벨이 높아질수록 낙하 속도 증가 + 동시 낙하 단어 수 증가
- 레벨 클리어 시 다음 스테이지 이동, 실패(바닥 도달 3회) 시 게임 오버

#### FR-03: 레벨별 단어장
- 레벨별 단어 데이터 내장 (JSON 파일 또는 JS 배열)
- **초급** (레벨 1~3): 2글자 단어, 20~30개 (예: 사과, 하늘, 공책)
- **중급** (레벨 4~7): 3~4글자 단어, 20~30개 (예: 컴퓨터, 사랑해요)
- **고급** (레벨 8~10): 5글자 이상 단어, 20~30개 (예: 대한민국, 프로그래밍)
- 총 단어 수: 약 100~150개

#### FR-04: 점수 시스템
- 단어 제거 성공: +글자 수 × 10점
- 콤보 보너스: 연속 성공 시 콤보 배수 적용 (x1 → x2 → x3 최대)
- 점수는 게임 화면 상단에 실시간 표시

#### FR-05: 타이핑 통계
- 게임 종료(클리어 또는 게임 오버) 후 결과 화면 표시
- 통계 항목: 총 타이핑 단어 수, 정확도(%), 평균 WPM(분당 단어 수), 최고 콤보
- 세션별 통계 (영구 저장 없음, 게임 종료 후 초기화)

#### FR-06: 소리 효과
- 단어 입력 성공: 청명한 효과음
- 단어 바닥 도달(실패): 실패 효과음
- 레벨 클리어: 클리어 효과음
- 게임 오버: 게임 오버 효과음
- 배경 음악 (선택 사항, 볼륨 조절 가능)

#### FR-07: 메인 메뉴
- 게임 시작 (레벨 선택 또는 처음부터)
- 단어장 보기 (리스트 모드 — 게임 없이 단어 확인)
- 설정 (볼륨 조절, 텍스트 크기)
- 종료

### 2.2 비기능 요구사항 (Non-Functional Requirements)

| 항목 | 요구사항 |
|------|----------|
| **플랫폼** | Windows 10+ / macOS 12+ (Electron 기반) |
| **프레임워크** | Electron + React (또는 순수 HTML/CSS/JS) |
| **한글 입력** | IME 조합 처리 (compositionstart/end 이벤트 활용) |
| **프레임레이트** | 게임 루프 60fps 목표 (requestAnimationFrame) |
| **패키징** | electron-builder로 exe/dmg 빌드 가능 |
| **파일 크기** | 설치 파일 200MB 이하 (사운드 에셋 포함) |

## 3. 성공 기준 (Success Criteria)

| ID | 기준 | 측정 방법 |
|----|------|----------|
| SC-01 | 레벨 1~10 전 스테이지 정상 플레이 | 수동 플레이테스트 |

| SC-05 | 소리 효과 4종 정상 재생 | 각 이벤트 트리거 테스트 |
| SC-06 | electron-builder로 실행 가능한 앱 빌드 | 빌드 후 앱 실행 확인 |

## 4. 범위 정의

### In Scope (구현 대상)

- Electron 앱 셸 + 개발 환경 셋업
- 게임 화면 (낙하 단어, 입력창, 점수 HUD)
- 레벨 스테이지 시스템 (1~10)
- 레벨별 단어장 (초급/중급/고급, 총 ~150개)
- 점수 시스템 + 콤보
- 타이핑 통계 결과 화면
- 소리 효과 (Web Audio API 또는 Howler.js)
- 메인 메뉴 + 설정 화면 (볼륨 조절)
- 단어장 리스트 모드

### Out of Scope (제외 항목)

- 온라인 랭킹 / 서버 연동
- 멀티플레이어
- 한글 자모 단위 연습 (낱자 타자)
- 영문 단어 지원
- 모바일 버전
- 자동 업데이트 시스템

## 5. 기술 스택 제안

| 계층 | 기술 | 선택 이유 |
|------|------|----------|
| 앱 셸 | **Electron v28+** | 크로스 플랫폼 데스크탑 앱 |
| UI 렌더러 | **React 18 + Vite** | 컴포넌트 기반 UI, 빠른 개발 |
| 게임 렌더링 | **Canvas API 또는 DOM** | 별도 게임 엔진 불필요한 수준 |
| 한글 입력 | **IME composition 이벤트** | 브라우저 내장 처리 |
| 사운드 | **Howler.js** | 경량 사운드 라이브러리 |
| 빌드 | **electron-builder** | exe/dmg 패키징 |
| 단어 데이터 | **JSON 내장 파일** | 외부 의존성 없음 |
